TCU 136檢定 - Strava 資料使用說明與隱私政策

最後更新日期：2025年12月24日

1. 關於我們

本服務由 憲動工作室有限公司（Taiwan Cycling Union） 營運， 用於支援 TCU 136 檢定活動（每年三月、六月、九月、十二月於台中大坑 136 自行車路線舉辦） 以及車友小比賽的成績計時與排名系統。

2. 我們如何使用 Strava 資料

當您授權連接 Strava 帳號時，我們會透過 Strava API 取得以下資料：

  • 基本資料：姓名、運動員 ID、頭像、使用者名稱
  • 活動資料：您在活動期間的騎乘紀錄（路線、距離、時間、速度、爬升等）

3. 資料使用目的

我們取得您的 Strava 資料，僅用於以下目的：

  • 驗證您在 136 檢定活動日完成官方路線
  • 計算您的完成時間與成績排名
  • 製作數位證書與活動成績公告
  • 支援車友小比賽的即時排行榜

我們絕不會：

  • 販售或轉售您的個人資料給任何第三方
  • 將您的資料用於廣告或行銷目的
  • 公開您不希望公開的個人隱私資訊

4. 資料保留期限

  • 活動成績：為了維護歷史紀錄與排名查詢，活動成績將永久保留
  • Strava 授權：您可以隨時取消授權，取消後我們將停止存取您的 Strava 新活動
  • 個人資料移除：若您要求刪除所有資料，請聯絡客服，我們會在 7 個工作天內完成刪除作業

5. 資料安全保護

您的資料存放於加密的雲端資料庫（Supabase），我們採用業界標準的安全措施， 包括 HTTPS 加密傳輸、防火牆、存取權限控管等，以保護您的個人資料不被未經授權的存取。

6. 您的權利

您隨時可以：

  • 取消 Strava 授權：在報名表單中點擊「取消授權」，或前往 Strava 設定頁面 移除授權
  • 查看您的資料：登入會員專區查詢您的報名紀錄與成績
  • 要求刪除資料：聯絡客服信箱，我們會協助您移除個人資料

7. 第三方服務聲明

本服務使用 Strava API 取得您的運動資料。 您的 Strava 帳號與資料由 Strava, Inc. 管理，請同時參閱 Strava 隱私政策。

8. 隱私政策變更

我們可能會不定期更新本隱私政策。重大變更時，我們會透過網站公告或電子郵件通知您。 建議您定期查看本頁面以了解最新內容。
